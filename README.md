# ReusedKey Attack on Stream Cipher

**Arguments:** "file with frequencies of chars" "file with cryptograms"

File with frequencies format:

````c
a : 10.0
b : 0.1
...
````

Cryptograms format (each cryptogram in one line):

```c
10101110 01010101 ...
10101010 10101010 ...
```

Attached freq.txt file contains simplified frequencies for Polish.
